# antrian-app


1. konfigurasi base_url : URL dengan link direktori project
    - buka file application/config/config.php
    ```php
      $config['base_url'] = 'URL';
    ```
2. untuk mengkonfigurasi koneksi database
    - buka file application/config/database.php
    ```php
      'hostname' => '127.0.0.1',
      'username' => 'root',
      'password' => '',
      'database' => 'antrian_app',
    ```
